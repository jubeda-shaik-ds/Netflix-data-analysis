# 🎬 Netflix Data Analysis

## 📌 Project Overview

This project focuses on analyzing a Netflix dataset using Python to extract meaningful insights about content distribution, trends, and patterns. It demonstrates core data analysis skills such as data cleaning, exploration, and visualization.

---

## 🎯 Objectives

* Analyze the distribution of Movies vs TV Shows
* Identify top countries producing content
* Understand trends in content release over the years
* Visualize insights using graphs

---

## 🛠️ Tools & Technologies

* **Python**
* **Pandas** – Data manipulation and analysis
* **Matplotlib** – Data visualization

---

## 📂 Dataset

The dataset contains information about Netflix titles including:

* Type (Movie/TV Show)
* Title
* Country
* Release Year

---

## 📊 Key Insights

* Movies are more frequent than TV Shows in the dataset
* The United States and India produce the most content
* Significant growth in content release after 2015
* Diverse range of content across multiple countries

---

## 📈 Visualizations

The project includes:

* Bar chart comparing Movies vs TV Shows
* Country-wise content distribution
* Release year trend analysis

---

## 🚀 How to Run the Project

1. Clone the repository
2. Install required libraries:

   ```bash
   pip install pandas matplotlib
   ```
3. Run the Python script:

   ```bash
   python netflix_analysis.py
   ```
